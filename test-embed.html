🌐 External Website - Campaign Embed Demo

This page simulates an external website (blog, creator site, etc.) embedding CrowdPay campaign widgets. The embeds below are live and will update in real-time when contributions are made.

How it works: The iframe loads from http://localhost:5173/embed/campaigns/:id and communicates with the parent page via postMessage for auto-resizing. No authentication required - anyone can view the embed!

Features:
  • ✅ Live progress bar updates via SSE (within 10 seconds)
  • ✅ "Back this campaign" button opens CrowdPay in new tab
  • ✅ Responsive design (300px - 600px width)
  • ✅ No authentication required
  • ✅ Auto-resizes via postMessage
  • ✅ Permissive CORS headers for cross-origin embedding
